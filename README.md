# Data Standardization System

데이터 표준화 지원 시스템은 조직 내 데이터의  
**용어(Terminology), 단어(Vocabulary), 도메인(Domain)** 를 관리하기 위한  
웹 기반 **데이터 표준 관리 시스템**입니다.

## Tech Stack
🐍 Python (FastAPI) | 🌐 HTML / JavaScript | 🗄 MariaDB | 🔧 Uvicorn | 🗂 GitHub

## Run
pip install -r requirements.txt  
uvicorn main:app --reload
